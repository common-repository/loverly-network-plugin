== Description ==

The Lover.ly Network Plugin allows publishing partners to connect their blog to the Lover.ly platform.


== Installation ==

1.	In an FTP program, open the `wp-content/plugins` folder
2.	Browse to your WordPress administration panel (http://www.example.com/wp-admin).
3.	Click the "Plugins" link in the top right menu.
4.	In the list of inactive plugins, check the box next to "Lover.ly Network Plugin", and click the "Activate" button.
5.	Click the "Lover.ly Network" link on your menu.
6.  You'll have one option, your settings, where you can choose to display block or inline.
7.  When you edit posts, you will now see the ability to fill in information.
